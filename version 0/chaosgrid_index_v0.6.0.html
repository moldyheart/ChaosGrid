ChaosGrid Help

Secure Mode uses window.crypto.getRandomValues and rejection sampling to reduce modulo bias when generating integers.

Seed Mode uses a deterministic PRNG for reproducible experiments. It is not suitable for security-sensitive use.

Human-like Mode detects visible patterns such as repeated values, long increasing/decreasing runs, fixed differences, high duplicate rate, and clustered distribution. It reduces obvious patterns but does not prove true randomness.

CSV and XLSX are suitable for Excel and data analysis. JSON is suitable for API mock and development. TXT is suitable for simple number lists or plain records.

This tool does not generate physical true randomness and should not be used as the only source for cryptographic keys or high-stakes security systems.
